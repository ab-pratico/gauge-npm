

# Install

```bash
npm i --save-dev gauge-npm
```

or with `yarn`

```bash
yarn add -D gauge-npm
```


# Setup a Gauge Project

```bash
gauge-setup && cd gauge-project && npm i
```

or with `yarn`

```bash
gauge-setup && cd gauge-project && yarn  && yarn-bin-fix
```
